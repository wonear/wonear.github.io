palette翻译中文意思为 调色板、主色调，涉及到颜色读取的需求，一般都能使用。

调色板取色库依赖

implementation ‘androidx.palette:palette-ktx:1.0.0’

这个库的内容很少，用起来也很简单，做下简单记录。

使用方式有两种，同步方式和异步方式，使用的方法在Palette类的顶部注释里

主要方法调用

  1. 获取图片中的颜色值getSwatches()

  1. 获取各个类别的颜色值

    这边获取到的值根据图片不同，获取到不一样的值，可能获取到空值，在赋值使用时需要做空判断和处理

private fun getColorList(palette: Palette): MutableList<ColorBean> {
    val colorList = mutableListOf<ColorBean>()

    colorList.add(ColorBean(palette.dominantSwatch, "占主导地位的样本--dominantSwatch"))//占主导地位的样本

    colorList.add(ColorBean(palette.mutedSwatch, "柔和的--mutedSwatch"))//柔和的
    colorList.add(ColorBean(palette.darkMutedSwatch, "柔和的暗色--darkMutedSwatch"))//柔和的暗色
    colorList.add(ColorBean(palette.lightMutedSwatch, "柔和的亮色--lightMutedSwatch"))//柔和的亮色

    colorList.add(ColorBean(palette.vibrantSwatch, "有活力的--vibrantSwatch"))//有活力的
    colorList.add(ColorBean(palette.darkVibrantSwatch, "有活力的暗色--darkVibrantSwatch"))//有活力的暗色
    colorList.add(ColorBean(palette.lightVibrantSwatch, "有活力的亮色--lightVibrantSwatch"))//有活力的亮色


    return colorList


}
